# 🚀 Full Stack Authentication System (Backend)

This is the backend service for a Full Stack Authentication System built using the MERN stack. It provides secure user authentication using JWT, password hashing, and protected routes.

---

## 🧠 Features

- User Registration with validation
- User Login with JWT authentication
- Password hashing using bcrypt
- Protected routes (authorization middleware)
- Rate limiting to prevent brute-force attacks
- Input validation (email & password)
- MongoDB database integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- bcryptjs
- express-rate-limit
- dotenv
- cors
